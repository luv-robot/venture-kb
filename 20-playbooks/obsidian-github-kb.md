# Obsidian + GitHub Knowledge Base Playbook

## Goal

Use GitHub as the source of truth and Obsidian as the daily interface for reading, editing, searching, and linking notes.

## Setup

1. Create a private GitHub repository.
2. Clone it locally.
3. Open the local repo folder as an Obsidian vault.
4. Use Markdown files as the primary knowledge format.
5. Commit and push regularly.
