
# Knowledge Base Guide

## Capture Flow

1. Capture rough material into `00-inbox/`.
2. Review and classify.
3. Convert stable conclusions into:
   - `30-decisions/`
   - `20-playbooks/`
   - `10-projects/<project>/`
4. Move raw or outdated material to `70-archive/`.

## Note Types

Use the `type` property:

- `project`
- `decision`
- `playbook`
- `research`
- `prompt`
- `question`
- `archive`

## Status Values

Use the `status` property:

- `active`
- `draft`
- `needs-review`
- `stale`
- `archived`
