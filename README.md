# Venture Knowledge Base

This repository is my source of truth for project knowledge, product ideas, agent workflows, research notes, and decision records.

## Purpose

This knowledge base is used to:

- Preserve important decisions
- Track project assumptions
- Maintain reusable playbooks
- Convert long conversations into structured notes
- Reduce repeated reasoning across projects
- Support future use by Hermes or other tools

## Main Areas

- `00-inbox/`: temporary capture area
- `10-projects/`: project-specific knowledge
- `20-playbooks/`: reusable operating procedures
- `30-decisions/`: decision records
- `40-prompts/`: reusable prompts
- `50-research/`: market, technical, and product research
- `70-archive/`: raw or deprecated material
- `90-templates/`: note templates
